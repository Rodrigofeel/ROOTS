### ROOTS

## Acomodações exclusivas em meio á naturaza

Roots é um projeto frontend, uma aplicação web que conecta acomodações em meio a natureza com seus clientes, que buscam experiências autênticas e imersivas.

## Arquitetura e Decisões Técnicas

Foi utilizado **Next.js** para aproveitar Server Components e melhor performance.
Os componentes forma separados em uma pasta dentro de **/app/components**.
As customizações e implementação de efeitos foram organizadas dentro da pasta **/hooks**.
